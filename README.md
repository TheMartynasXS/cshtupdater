Custom Skin hash table updater

Program thats downloads league of legends hashes into currect folder every 7 days from initial run day.

1. Put in the location you want hashes to be downloaded for example:
D:\Obsidian_5.1.7\hashes
2. Then create a shortcut by rightclicking on the cshtupdater.exe
3. Press windows key + R and enter shell:startup
4. Then put that shortcut in the startup folder. 

Notes: 
  When using with Obsidian disable hash sync option.
  Disclaimer: I don't know yet how to build releases with C# so the executable is added manually temporarily.
Enjoy updated hashes.
